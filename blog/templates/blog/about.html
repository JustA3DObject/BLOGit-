Hello there visiter!

You have landed on a simple blogging website created by:
Aayush Vishnoi.
This website is created using Django 3.2 framwork and is written using Python and HTML with some Javascript and a pinch of CSS.

Some key points:

  • You can create POSTS and publish them for everyone else to see, or you can save them as drafts to publish later.
  • You can edit POSTS after publishing or while they aree in drafts.
  • Viewers can COMMENT on your POSTS and you can APPROVE or REMOVE those COMMENTS.
  • You can DELETE your posts after PUBLISHING as well.
  • Editing POSTS made easier with options like BOLD, UNDERLINE, ITALIC, HREF along with H2 size, H3 size & Block-quote.
  • Images are not included yet but will be rolled out in near future.
  • Only superusers can login and create POSTS right now, user login will roll out soon.

Thanks for checking it out :)
